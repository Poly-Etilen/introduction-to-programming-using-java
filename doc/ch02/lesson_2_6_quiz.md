## 객관식 문제

a) JDK는 무료이고 JRE는 유료다

### 2. Java 소스 파일의 확장자는?
a) .class
b) .java
c) .jar
d) .jdk

a) java HelloWorld.java
b) javac HelloWorld
c) javac HelloWorld.java

a) java HelloWorld.class
b) java HelloWorld
c) javac HelloWorld

a) Ctrl+S
