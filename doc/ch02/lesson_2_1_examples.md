```java
        // 1. 단일 행 주석: // 로 시작
        // 2. 다중 행 주석: /* 로 시작하고 */로 끝
        // 3. 문서화 주석: /** 로 시작하고 */로 끝 (클래스 위에 이미 사용)

           설명을 작성할 수 있습니다. */
        // "세 가지 주석 유형을 사용했습니다." 출력
        
        // int temp = 100;  // 이 코드는 임시로 비활성화됨
        
        /* 
         * 보기 좋게 정렬된
         * 다중 행 주석
         */

        // 여기에 코드를 작성하세요
    }  // main 메소드 끝
    
}  // CommentDemo 클래스 끝
```


## 4. 특수 문자 출력 예제

### 예제 4-1: 이스케이프 문자
```java
public class EscapeCharacters {
    public static void main(String[] args) {
        // TODO: 이스케이프 문자를 사용하여 특수 문자를 출력하세요
        // 힌트:
        // 1. 큰따옴표 출력: \"를 사용
        //    - "그가 말했다: \"안녕하세요!\""
        // 2. 역슬래시 출력: \\를 사용
        //    - "파일 경로: C:\\Program Files\\Java"
        // 3. 탭 문자로 표 만들기: \t 사용
        //    - 헤더: "이름\t나이\t직업"
```

## 실습 과제

1. **자기소개 프로그램**: 자신의 이름, 나이, 취미를 출력하는 프로그램 작성
2. **좋아하는 인용구**: 좋아하는 명언을 큰따옴표를 포함하여 출력

4. **시간표 출력**: 탭을 사용하여 일주일 시간표 출력
5. **로고 디자인**: 회사나 팀의 텍스트 로고 디자인
